{% endblock %} {% block problem_body %}

Let's look at an example of chaos in nature, the logistic map. It's a simplified model used to describe population sizes and can be written mathematically as $x_{n+1} = r*x_n(1 - x_n)$ where $0 < r < 4$ is a constant and $0 < x_n < 1$ represents the population size. Starting with an initial population size of $x_0 = 0.5$, your program should read in a value for $r$ then calculate and print out the first 50 values of the logistic map $x_0, x_1, x_2, \dots, x_49, x_50$ on separate lines. You should see different behavior depending on the value of $r$.

{% endblock %} {% block input_description %}

The value of $r$.

{% endblock %} {% block input_example %} 2.81 {% endblock %} {% block output_description %}

The first 50 values of the logistic map $(x_0, x_1, x_2, \dots, x_49, x_50)$ on separate lines, with at least 6 decimal place accuracy.
