They're tiny circuits that work with 0's and 1's and perform basic boolean operations, usually with 2 inputs.



A simple logic gate would be the NOT gate, which inverts the input so a 0 becomes a 1 and vice versa. An AND gate produces 1 only if both of its inputs are 1, otherwise it produces 0.

The NAND gate is a an AND gate followed by a NOT gate, so it's performing a (NOT AND) operation. Try to write a function NAND(A, B) that returns the output of a NAND gate.

Notes

  • All logic gates are made using transistors so transistors are the building blocks of all phones and computers.
  • The NAND gate (and NOR gate) is special in that it's a universal gate so you can build any other gate using a NAND gate. You could even build a CPU completely out of NAND gates, although it would probably be bigger and slower than if you just used the actual gates you needed.
