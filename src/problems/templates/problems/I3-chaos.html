Let's use loops to look at an example of chaos in nature, the logistic map. It's a dumbed down version of a model used to describe population sizes and can be written mathematically as $x_{n+1} = r*x_n(1 - x_n)$ where $0 < r < 4$ is a constant and $0 < x_n < 1$ represents the population size. Starting with $x_0 = 0.5$, your program should read in the input value $r$ then calculate and print out the first 50 values of the logistic map $x_0, x_1, x_2, \dots, x_49, x_50$ on separate lines.
